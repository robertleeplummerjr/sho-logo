# SHO Logo

An svg generator for the 92-95 Ford Taurus SHO slotted logo generator.

![SHO Logo Rendered](rendered.png)

## Overview

This project provides a web-based tool to generate an accurate SVG version of the slotted SHO logo found on 1992-1995 Ford Taurus SHO models.

## Usage

1. Open `logo.html` in a web browser.
2. The logo will be rendered on the page.
3. Click on the rendered logo to download the `sho-logo.svg` file.
4. Press 'D' on your keyboard to toggle debug mode, which reveals the geometric primitives used to construct the logo.
